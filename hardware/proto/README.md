# KappashaOS Protofolder
Early prototype. Not for real skin. Yet.

## Overview
/proto/ contains cyberpunk experiments: moto_pixel (ink.rs), postcard effect (postcard.frag), flat honesty (grokflat.frag). Pixels crawl to match gaze, screens breathe truth, tattoos pulse green on eye contact. Apache 2.0, revocable for misuse.

## Components
- **ink.rs**: Photolitho tattoo, 0.2 micron grooves, retro-reflective gaze, bone conduction on green key.
- **postcard.frag**: Green for keyed gaze, red for unread, multi-user pixel split.
- **grokflat.frag**: Flat honesty, no parallax, green kappa spiral for perspective.
- **ink_sim.py**: Numpy sim for multi-user gaze, 5 users, theta spiral.

## Usage
Run demos:
```bash
python3 ink_sim.py
# WebGL for postcard.frag, grokflat.frag in browser
```
